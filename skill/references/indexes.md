# Indexes

Indexes are optional but useful. They make the vault navigable and help the agent avoid duplicate notes.

## Recommended Files

- `Index - Global.md`
- `Index - Concepts.md`
- `Index - Pivots.md`
- `Index - Frameworks.md`
- `Index - HowTo.md`
- `Index - Maps.md`

Configure their location in `config.yml`.

## Entry Format

```markdown
- [[Note title]] - concept - domain/topic - short mechanism - intermediate
```

Fields:

- wikilink
- type
- domain
- short description
- grade or status

Recommended grades:

- `fragile`: promising but thin
- `intermediate`: useful and structured
- `dense`: mature, enriched repeatedly
- `legacy`: older reference retained for context

Do not mark a new note as dense on creation.
